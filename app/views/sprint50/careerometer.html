Compare different jobs

You may want to compare one job you’re interested in with another to see which one you prefer. You can do this using the Careerometer. Use the Careerometer to compare information like:

  • salary
  • working hours
  • how many jobs there are likely to be in this area
  • which industries you can find these job roles in
This tool is not suitable for users with screen readers. We are not able to make this tool accessible as the Careerometer is supplied directly from LMI For All. If you would prefer to use the tool with the help of one of our career advisers, please contact us.

How to use the Careerometer

To compare two different jobs, you should:

  • click the dotted square
  • type in the first job that you think you might be interested in
  • select from the drop-down list
  • repeat the steps to add your second job

If you want to compare a job you’re interested in against the national average, you should:

  • click the dotted square
  • type in the job that you think you might be interested in
  • select from the drop-down list
  • click “display UK average” on the second card
